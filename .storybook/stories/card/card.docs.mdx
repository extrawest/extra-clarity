## Description

A card presents high-level information and can guide the user toward related actions and details.

## Quick Start

> **Standalone component**: yes, but exported also within a module-like array containing additional directives

```typescript
import { EcCardModule } from '@extrawest/extra-clarity/card';
```

{/* prettier-ignore */}
```html
<ec-card
  [title]="title"
  [empty]="isEmpty"
  [loading]="isLoading"
  [error]="error"
  [spinnerSize]="'md'"
>
  ...

  <ec-card-footer>
    ...
  </ec-card-footer>
</ec-card>
```
