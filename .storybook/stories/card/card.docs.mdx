# Card

A card presents high-level information and can guide the user toward related actions and details.

<br/>

### Import module

```typescript
import { EcCardModule } from "@extrawest/extra-clarity/card";
```

### Code block

```html
<ec-card
  [title]="title"
  [empty]="empty"
  [loading]="loading"
  [error]="error"
>
  ...

  <ec-card-footer>
    ...
  </ec-card-footer>
</ec-card>
```
