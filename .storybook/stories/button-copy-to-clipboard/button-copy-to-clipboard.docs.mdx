## Description

This component was designed to simplify copying string values to the clipboard.

### Configuration

There are multiple inputs to configure the component.

The only one required is `[textToCopy]` containing the actual string to be copied when the button is pressed.
If the value is nullish or not provided at all, the button is disabled.

When the value is copied, the `(copied)` output emits an event,
which may be used for providing an external notification about successful copying.

Please refer to the [API](#api) section below for more details about the inputs and outputs of this component.
